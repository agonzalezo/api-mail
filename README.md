# api-mail
An API REST to send massive mails using templates.

## Installation
- npm i
- You need this env vars.
    ```bash
    API_LOG_LEVEL='short'
    API_NAME='mail'
    API_MAIL_KEY='XXxx'
    API_MAIL_USER='XXxx@gmail.com'
    NAME_SPACE='xylon'
    SMTP_HOST='smtp.gmail.com'
    SMTP_PORT='465'
    PORT=3001

## Usage
- npm start
